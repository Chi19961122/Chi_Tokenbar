TokenBar 分享卡片 — 六樣態重設計預覽

這是「分享」功能六張可分享用量卡的全面重設計提案,實作前先用純 HTML 預覽。每個樣態同時呈現重設計後的 16:9(1200×675,縮放至約 600px 寬)與 9:16(360×640 原尺寸),左右並排,使用真實硬編數據。

設計方向沿用 App 的極簡編輯部語彙:zinc 灰階 + #EC4899 洋紅點綴、強烈焦點數字、tabular 對齊數字、letterspaced 大寫微標籤,並跨六卡統一頁尾署名系統(電池標記 + TokenBar + 大寫日期)。嚴守 §0 隱私鐵則 —— 全程無任何專案名稱。

字型註記:Geist / Geist Mono / Playfair Display 在獨立檔中無法載入,此預覽以系統字近似(Segoe UI / Consolas / Georgia)。真機請換回專案 public/fonts/ 的 Geist bundle,字重與字距會更緊緻。此檔不修改任何 repo 檔案。
