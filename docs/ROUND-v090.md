# ROUND v0.9 — 全面改名 TokenBar → Atoll(2026-07-19 使用者定案)

> 使用者定案:產品**全面更名 Atoll**,且 **v0.8.0 直接以 Atoll 之名出貨**(壓住 0.8.0 釋出,先改名再打包)。名字來源:環礁(atoll)= 環/礁湖母題,呼應額度環形儀表。品牌記號 = ◎ 同心環(取代電池)。accent 維持洋紅(亮 #EC4899 / 暗 #F472B6)。

## 已拍板決定

- **改名深度**:全面(exe/安裝檔/系統匣/視窗標題/package/Cargo/identifier/i18n/header/分享卡)。
- **設定資料夾**:`%APPDATA%\TokenBar\settings.json` → `%APPDATA%\Atoll\settings.json`,**加一次性遷移**(啟動時若 Atoll 檔不存在但 TokenBar 檔存在 → 複製一次)。設定路徑是硬編碼(config.rs),非 identifier 衍生,故改 identifier 本身不動設定。
- **App 圖示**:定案 **C · 額度弧**(深底圓角方 + 洋紅 62% 額度弧 + 中心礁湖點,呼應額度儀表功能;canvas 生成邏輯見對話,1024² → `icon-source.png` → `npm run tauri icon` 重生各尺寸)。候選比較板在 `design/refs/atoll-icon-options.png`。
- **分享卡**:**連新方向一起上**(採 `design/refs/atoll-share-preview.html` 的六款:Atoll 環礁儀表 / Ledger / Terminal / Minimal / Sounding / Seal),非只換名。旗艦 = **Atoll 環礁儀表**(取代 island_card 為預設樣式);Sounding(測深)取代舊 fuel 卡槽。全部戴 ◎ 署名 + Atoll。
- **header 品牌**:窄面板塞不下「Atoll」字標(實測會把頁籤壓兩行),故 **header 只放 ◎ 環記號**;Atoll 字標留給視窗標題/系統匣/設定/分享卡。
- **bundle identifier**:`com.qqq01.tokenbar` → `com.qqq01.atoll`。副作用:新安裝檔不會就地升級舊 TokenBar 安裝,會並存;使用者手動移除舊 TokenBar 一次(個人工具可接受)。無單例 mutex 顧慮(未發現)。

## 風險/待議

- **identifier 改變 → 安裝並存**:新 Atoll 安裝不會覆蓋舊 TokenBar,使用者需手動移除舊版一次。真機驗收時確認。
- **設定遷移**:遷移邏輯要在「讀設定之前」跑;若兩檔都不存在=全新使用者,正常走預設。verifier 要測「有舊無新→複製」「有新→不覆蓋」「都無→預設」三態。
- **build:release exe 名**:productName/binary 改 atoll 後,collect-installers 的 exe 路徑與 taskkill 目標必須同步為 atoll.exe,否則打包抓不到產物或殺錯行程。這是 T-920 的核心風險點。
- **分享卡新方向**:Sounding(測深)海圖曲線在真資料下的觀感、Atoll 環礁儀表 SVG 弧的縮放,需真機/mock 驗。
- **User-Agent 改 atoll**:cosmetic;不影響限流(限流走 client_id 非 UA)。低風險。
